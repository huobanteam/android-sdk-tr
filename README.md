# android-sdk-tr
##Usage
    HBApiImpl hbApi = HBApiImpl.getInstance(this);
###OPEN_SPACE(打开工作区)
    hbApi.openSpaceWithId("");
###OPEN_ITEM(打开表格详情)
    hbApi.openItemWithId("");
###OPEN_TABLE(打开表格)
    hbApi.openTableWithId("");
###OPEN_APP_WITH_TABLE_MESSAGE(打开应用)
    hbApi.openAppWithHBTableMessage("");
